## Getting started

# Running the project locally

1. Clone the repo

```
git clone https://github.com/LisaPisa12/femtasy-tech.git
cd femtasy-tech/femtasy-tech
```

2. Install dependencies in client folder

```
yarn install
```

3. Start it up

```
'yarn start' in the client folder .
```
